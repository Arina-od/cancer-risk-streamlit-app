# 🩺 Cancer Risk Prediction App

Учебный проект по машинному обучению и разработке веб-приложений.

Приложение позволяет оценить риск развития различных видов онкологических заболеваний на основе индивидуальных факторов здоровья и образа жизни.

## 📌 О проекте

В рамках проекта были разработаны модели машинного обучения, которые оценивают риск для следующих типов рака:

- рак молочной железы  
- рак простаты  
- рак кожи  
- рак толстой кишки  
- рак лёгкого  

Для каждого типа используется отдельная модель.

Пользователь заполняет анкету, после чего приложение рассчитывает и отображает:
- оценку риска (в процентах)
- категорию риска (низкий / умеренный / высокий)
- рекомендации по снижению риска

## ⚙️ Используемые факторы

Модели учитывают:

- возраст и индекс массы тела (ИМТ)  
- пол  
- семейную историю онкологических заболеваний  
- мутацию BRCA  
- инфекцию *H. pylori*  
- образ жизни (курение, алкоголь, питание, физическая активность)  
- факторы окружающей среды и работы

## 🚀 Запуск проекта локально

Установить зависимости:

```bash
pip install -r requirements.txt
```bash

Обучить модели:

```bash
python train_models.py
```bash

Запустить приложение:

```bash
streamlit run app.py
```bash
